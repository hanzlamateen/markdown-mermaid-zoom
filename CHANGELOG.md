# Changelog

## 0.1.0 - Initial Release

- Mermaid diagram rendering in VS Code's built-in markdown preview
- Zoom in/out via buttons and mouse scroll
- Pan support with click-and-drag
- Fullscreen mode for diagrams
- Vertical resize by dragging bottom edge
- Configurable light/dark mode themes
- Configurable mouse navigation modes
- Support for `:::` mermaid blocks
